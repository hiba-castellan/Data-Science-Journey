ClinicalForecast — Early Sepsis Prediction System

A production-grade, real-time ICU risk scoring system that predicts sepsis onset hours before clinical deterioration — combining structured time-series data with LLM-extracted signals from free-text nursing notes.

Multimodal clinical AI: structured EHR time-series + LLM-extracted unstructured nursing note signals, deployed end-to-end.

Core modules

Module 1
Data Pipeline
MIMIC-IV ingestion, irregular time-series handling, MICE imputation, feature store
Module 2
Predictive Modeling
LightGBM + LSTM ensemble with Cox PH survival model for time-to-sepsis
Module 3
Explainability
SHAP force plots per patient — why is this patient flagged?
Module 4
Deployment
FastAPI + Docker on HuggingFace Spaces with live risk scoring endpoint

Key novelty — NLP fusion layer

BioMistral-7B parses free-text nursing notes (MIMIC-IV) to extract symptom signals and clinical urgency scores — temporally aligned to vitals windows and fused as features into the ensemble. Most published sepsis models use only structured data; this bridges the gap.
